# Assets & Attribution\n\n## Campaign Assets\n\nThis campaign is designed to work with minimal external assets initially. Phi-4-mini can generate detailed descriptions that evoke visual and audio atmosphere without requiring pre-commissioned art.\n\n## Optional Asset Directories\n\nIf the author creates visual or audio assets, organize them as:\n\n```\nassets/\n├── art/\n│   ├── characters/\n│   ├── locations/\n│   └── objects/\n├── audio/\n│   ├── ambient/\n│   ├── music/\n│   └── sfx/\n└── video/\n    └── cinematics/\n```\n\n## Attribution Guidelines\n\n- **Human-Created**: Mark with author name and creation date\n- **AI-Generated**: Mark with `generated: true`, model name, and human refiner (if applicable)\n- **Third-Party**: Include license and attribution per source requirements\n- **Derivative**: Clearly indicate which work(s) inspired this asset\n\n## Example Format\n\n```yaml\nfile: tower_entrance.png\ncreated: 2026-02-15\nauthor: \"Jane Author\"\ntype: \"concept art\"\nlicense: \"CC-BY-4.0\"\n```\n\n```yaml\nfile: leory_voice_sample.wav\ngenerated: true\nmodel: \"tts-neural-v3\"\nbase_voice: \"raspy-mentor\"\nrefiner: \"Jane Author\"\nlicense: \"CC-BY-4.0\"\n```\n\n---\n\n**Note**: The Wizard Runner campaign runs fully without external assets. Art and audio enhance immersion but are not required for core gameplay.\n"
